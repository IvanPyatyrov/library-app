# Библиотечное приложение на Python

Это консольное приложение для управления библиотекой книг. Оно позволяет добавлять, удалять, искать и отображать книги.

## Функционал

1. **Добавление книги**: Пользователь вводит название книги, автора и год издания. Книга добавляется с уникальным идентификатором и статусом "в наличии".
2. **Удаление книги**: Пользователь может удалить книгу по уникальному идентификатору.
3. **Поиск книги**: Пользователь может искать книги по названию, автору или году издания.
4. **Отображение всех книг**: Приложение выводит список всех книг с их идентификаторами, названиями, авторами, годами и статусами.
5. **Изменение статуса книги**: Пользователь может изменить статус книги на "в наличии" или "выдана".

## Использование

1. Скачайте или клонируйте репозиторий.
2. Убедитесь, что Python установлен на вашем компьютере.
3. В терминале выполните команду:

   python app.py


## Тестирование
Чтобы протестировать приложение, выполните команду:

python -m unittest test_library.py
